# Flux RSS des actualités du SDIS 38

Ce dépôt génère automatiquement un fichier `rss.xml` depuis la page officielle des actualités du SDIS 38.

## Adresse du flux

Après activation de GitHub Pages :

`https://remy1838.github.io/sdis38-rss/rss.xml`

## Première utilisation

1. Téléverser tous les fichiers dans le dépôt GitHub `sdis38-rss`.
2. Ouvrir **Settings > Actions > General**.
3. Dans **Workflow permissions**, sélectionner **Read and write permissions**, puis enregistrer.
4. Ouvrir l'onglet **Actions**.
5. Sélectionner **Mise à jour du flux RSS SDIS 38** puis **Run workflow**.
6. Vérifier que le fichier `rss.xml` apparaît.
7. Dans **Settings > Pages**, publier la branche `main`, dossier `/ (root)`.
